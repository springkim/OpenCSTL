cstl_deque

Double-ended queue with amortized O(1) push/pop at both ends and random access. Equivalent to std::deque<T>.

SequenceRandom Accessdeque.h

Declaration

without USE_CSTL_FUNC
DEQUE(int) d = cstl_deque(int);
// Subscript access
d[i]
with USE_CSTL_FUNC
#define USE_CSTL_FUNC
DEQUE(int) d = new_deque(int);
// Subscript access
d[i]
ℹ

DEQUE(TYPE) supports d[i] subscript access just like vector. Unlike std::deque in C++, OpenCSTL's deque uses a contiguous-like circular buffer strategy that allows direct indexing with minimal overhead.

Example

c
#define USE_CSTL_FUNC
#include "opencstl.h"

void example_deque(void) {
    DEQUE(int) d = new_deque(int);

    push_back(d, 1);
    push_front(d, 2);
    push_back(d, 3); // [2, 1, 3]

    for(size_t i = 0; i < size(d); i++) {
        printf("%d ", d[i]);
    }

    pop_front(d);
    destroy(d);
}
