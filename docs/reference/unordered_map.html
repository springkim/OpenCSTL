cstl_unordered_map

Hash-based key-value store with O(1) average time complexity. Equivalent to std::unordered_map<K,V>.

UnorderedHash Tablehashtable.h

Declaration

without USE_CSTL_FUNC
UMAP(int) m = cstl_unordered_map(int, int);
with USE_CSTL_FUNC
#define USE_CSTL_FUNC
UMAP(int) m = new_unordered_map(int, int);
ℹ

The iterator (KEY_TYPE *it) points to the key. Access the associated value with cstl_value(it, VALUE_TYPE). Internally each bucket stores [key | value] contiguously.

Example

c
#define USE_CSTL_FUNC
#include "opencstl.h"

void example_umap(void) {
    UMAP(int) m = new_unordered_map(int, char*);

    insert(m, 1, "one");
    insert(m, 2, "two");

    int *it = find(m, 1);
    if (it) {
        printf("Key 1 has value: %s\n", cstl_value(it, char*));
    }

    destroy(m);
}
