cstl_list

Doubly linked list supporting constant time insertion and removal from anywhere. Equivalent to std::list<T>.

SequenceBidirectional Iteratorlist.h

Declaration

without USE_CSTL_FUNC
LIST(int) l = cstl_list(int);
with USE_CSTL_FUNC
#define USE_CSTL_FUNC
LIST(int) l = new_list(int);
ℹ

Iterators in cstl_list are pointers to the data (TYPE*). Unlike vector, elements are not contiguous in memory, but the library manages the node pointers internally so you can use cstl_next() and cstl_prev() to traverse.

Example

c
#define USE_CSTL_FUNC
#include "opencstl.h"

void example_list(void) {
    LIST(int) l = new_list(int);

    push_back(l, 10);
    push_front(l, 20);
    push_back(l, 30); // 20, 10, 30

    // traversal
    for (int *it = begin(l); it != end(l); it = next(it)) {
        printf("%d ", *it);
    }

    destroy(l);
}

Notes

✓

Iterators in cstl_list remain valid even after other elements are inserted or removed, as long as the element itself is not deleted.

⚠

cstl_list does not support subscript access (l[i]). Use vector or deque if you need random access.
